# 🚀 Painel NextLevel

O **Painel NextLevel** é uma aplicação web Full Stack simplificada, focada na gestão de produtividade e organização de treinos. O projeto demonstra o domínio de fluxos cliente-servidor, manipulação dinâmica de dados e persistência no navegador.

## 🛠️ Tecnologias Utilizadas

* **Backend:** Node.js e Express (Servidor e roteamento).
* **Frontend:** HTML5, CSS3 (Design moderno/Dark Mode) e JavaScript Vanilla (ES6+).
* **Persistência:** LocalStorage para salvamento de estados e status de conclusão.
* **Arquitetura:** Comunicação entre frontend e backend para carregamento de dados do usuário.

## 🌟 Funcionalidades de Destaque

-   **Lógica de Check/Uncheck:** Persistência do status de "concluído" mesmo após atualizar a página (F5).
-   **Gestão de Arrays Dinâmicos:** Uso de métodos como `.filter()` e `.find()` para manipulação de listas de tarefas e treinos.
-   **UX & Acessibilidade:** -   Atalhos de teclado (Enter) para adicionar itens sem uso do mouse.
    -   Foco automático (`.focus()`) no campo de texto para agilizar a rotina.
-   **Sistema de Autenticação:** Verificação de segurança para acesso ao painel.

## 🏁 Como Executar o Projeto

Para rodar o projeto localmente, você precisará ter o [Node.js](https://nodejs.org/) instalado.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/brunomarques280/NextLevel.git](https://github.com/brunomarques280/NextLevel.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor:**
    ```bash
    node server.js
    ```
4.  **Acesse no navegador:**
    `http://localhost:3000`
